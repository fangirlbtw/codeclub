More on fonts

To change all the fonts on the page - except <code> blocks. (like this page).

Put this in the <style> block:


	*{font-family:cursive;}
	code {font-family:monospace;}

More information on fonts here: http://www.w3schools.com/cssref/css_websafe_fonts.asp
