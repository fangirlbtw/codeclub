Changing text

Changing every time

This header is in a different font

For example, to get all <h4> headers to change to look like the header above, put this in the <style> block:


	h4 {
		font-size:36px;
		text-decoration:underline;
		font-family:impact, sans-serif;
	}

Changing just one

If you want to change just one thing...

For example to get this line of text to change colour, but no other.

Put this in the <style> block:


	#highlight {
		color:red;
	}

And write your line of text like this:


	<p id="highlight">For example to get this line of
	 text to change colour, but no other.</p>

Changing just more than one, but not every

If you want to change a few things, but not everything, like this:

    This is large

    This is large

    This is not large

Put this in the <style> block:


    .large {
        font-size:24px;
    }

And write your lines of text like this:


    <p class="large"> This is large </p>
    <p class="large"> This is large </p>
    <p> This is not large </p>
